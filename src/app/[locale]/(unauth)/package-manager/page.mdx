# 前端依赖管理

## npm config

```bash
npm config list
# show all defaults
npm config ls -l
npm config set registry https://registry.npmjs.org/
```

### 使用 monorepo

创建 pnpm-workspace.yaml

```yaml
packages:
  # all packages in direct subdirs of packages/
  - 'packages/*'
  # all packages in subdirs of components/
  - 'components/**'
  # exclude packages that are inside test directories
  - '!**/test/**'
```
