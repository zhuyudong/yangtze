# package.json

## package.json 中字段说明

### packageManager

规定项目中使用的 [pnpm](https://pnpm.io/) 版本，在 [zeabur](https://zeabur.com/docs/zh-CN/guides/nodejs) 部署时可用

```json
{
  "packageManager": "pnpm@8.15.5"
  //...
}
```

### engine

```json
{
  "engines": {
    "node": "20.11.1"
  }
  //...
}
```

如当前 Node.js 版本(18.19.1)和上述 engines 不匹配时，警告如下

```bash
WARN  Unsupported engine: wanted: {"node":"20.11.1"} (current: {"node":"v18.19.1","pnpm":"8.15.5"})
```

## 如何排序 package.json？

使用 [sort-package-json](https://github.com/keithamus/sort-package-json)

```bash
pnpm i sort-package-json -D
./node_modules/.bin/sort-package-json
```
