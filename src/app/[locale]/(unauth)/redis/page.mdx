## Redis

### Redis 基本命令

进入 redis REPL 环境

```sh
redis-cli -h localhost -p 6379 -a LEiMPnJBReHCZpxr
```

redis 基本命令

```sh
localhost:6379> keys *
(empty array)
localhost:6379> PING
PONG
localhost:6379> ping
PONG
localhost:6379>
```

### 宝塔机中的 Redis

通过软件商店安装的 Redis

```sh
# 启动redis
/etc/init.d/redis start
# 停止redis
/etc/init.d/redis stop
# 重启redis
/etc/init.d/redis restart
# 重载redis
/etc/init.d/redis reload
```

### 如何进入 redis docker 容器

```sh
$ docker exec -it redis-local /bin/bash
root@aeb61477521e:/data# redis-cli
127.0.0.1:6379> keys *
3) "session:6234a6f1-7c72-4229-b785-27f29048f535"
> HGET SCHEDULER_SWITCH TaskMonitor
> HSET SCHEDULER_SWITCH TaskMonitor '1'
127.0.0.1:6379> quit
root@aeb61477521e:/data# exit
exit
```

选项说明
`docker exec -it redis-local /bin/bash` 进入 redis 容器，
`redis-cli` 连接当前默认 redis 服务
