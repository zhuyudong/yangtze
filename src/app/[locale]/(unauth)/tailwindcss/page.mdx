API:

- `sr-only`

```css
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border-width: 0;
}
```

- `shrink-0`

```css
.shrink-0 {
  flex-shrink: 0;
}
```

- `border-transparent`

```css
.border-transparent {
  border-color: transparent;
}
```
