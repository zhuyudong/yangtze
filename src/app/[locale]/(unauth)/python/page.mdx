### 如何测试 `isinstance()` 和 `type()` 的执行性能

```bash
# Python 3.11.8

$ python -m timeit -s "variable = 'hello'" "isinstance(variable, str)"

$ python -m timeit -s "variable = 'hello'" "isinstance(variable, str)"
20000000 loops, best of 5: 14.9 nsec per loop
```
